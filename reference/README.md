# 参考代码 (Reference Code)

本目录包含原始的参考项目代码，用于学习和对比。

## python-patterns/

原始项目：https://github.com/faif/python-patterns

这是一个收集 Python 设计模式的项目，包含 23+ 种设计模式的简洁实现。

**用途**：
- 作为学习参考
- 对比 `improved-patterns/` 目录中的改进实现
- 了解经典的 Python 设计模式实现方式

**注意**：
- 此代码仅用于参考，不会被项目运行时使用
- 改进版的代码位于 `improved-patterns/` 目录
- 原始仓库的完整历史可在 GitHub 上查看
